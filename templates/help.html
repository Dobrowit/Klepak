Pomoc API

Endpoint: /upload

Metoda: POST

Opis: Endpoint służy do przesyłania danych wraz z obrazem. Dane są zapisywane na serwerze.

Body (JSON):

{
    "data": "2021-08-06",
    "opis": "Opis zdjęcia",
    "zdjecie": "base64kodowanyObraz",
    "latitude": 54.7578,
    "longitude": 17.5610
}

Odpowiedź:

{
    "message": "Dane zapisane pomyślnie"
}

Endpoint: /data

Metoda: GET

Opis: Endpoint zwraca wszystkie zapisane dane w formacie JSON.

Odpowiedź:

[
    {
        "data": "2021-08-06",
        "opis": "Opis zdjęcia",
        "zdjecie": "data/2021-08-06.jpg",
        "latitude": 54.7578,
        "longitude": 17.5610
    },
    ...
]
